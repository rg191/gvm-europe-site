# Publikálásra kész cikkek

Ez a mappa a **megjelenésre szánt, letisztított** cikkeket tartalmazza — csak cím, alcím és
a cikktörzs. A gyártási segédanyagok (szerkesztői jegyzet, Midjourney-promptok, „minta-deliverable"
keretek) a `docs/` alatti munkapéldányokban maradnak, ide nem kerülnek.

## Konvenció
- **Munkapéldány:** `docs/sample-article-*.md` — teljes deliverable, belső jegyzetekkel és
  vizuál-promptokkal.
- **Publikálásra kész:** `docs/published/*.md` — csak az olvasó elé kerülő szöveg.

## Megjelenés előtti emberi lépések (nem az ágens dönti el)
1. **Borítókép legyártása.** A munkapéldány Midjourney v7 promptját le kell renderelni tényleges
   képpé, és beilleszteni / a megjelenési felületre feltölteni.
2. **Korrektúra.** Egy emberi átolvasás.
3. **Byline és felület.** Szerző és megjelenési platform rögzítése.
